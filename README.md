# 📘 Scratch‑Projekte – Jungblut Kurs  
Sammlung aller selbst nachgebauten Scratch‑Projekte aus dem Jungblut‑Kurs

Dieses Repository enthält **alle meine selbst rekonstruierten .sb3‑Dateien**, die auf den Aufgaben aus dem Scratch‑Kurs von **Daniel Jungblut** basieren.  
Die Projekte wurden **von Grund auf neu erstellt**, erweitert, verbessert und teilweise technisch optimiert, um sie für Unterricht, Schülerfirmen, Workshops und Lernzwecke nutzbar zu machen.

> ⚠️ Hinweis  
> Dieses Repository enthält **keine Originaldateien** des Jungblut‑Kurses.  
> Alle Projekte wurden **eigenständig nachgebaut**.

---
## 🛠️ Technische Hinweise

- Alle Dateien liegen im **Scratch‑3‑Format (.sb3)** vor  
- Kompatibel mit:
  - Scratch Online Editor  
  - Scratch Desktop  
  - TurboWarp  
- Jede Datei wurde **getestet** und ist **fehlerfrei ladbar**

---

## 📦 Nutzung

### Scratch Online
1. Öffne: https://scratch.mit.edu/projects/editor  
2. Datei → „Hochladen von deinem Computer“  
3. `.sb3` auswählen

### TurboWarp
1. Öffne: https://turbowarp.org/editor  
2. `.sb3` Datei hineinziehen oder über Datei öffnen
